# gas-pwa-installer
Aplikasi Manajemen Administrasi Sekolah/Madrasah. Fitur: Administrasi (Profil Lembaga, Mapel, Data Induk), Akademik (Olah Nilai/Prestasi, Cetak Dokumen/List Siswa, Presensi Online), dan Layanan (E-Arsip, Laporan, Buku Tamu, Ketertiban/BK, Modul Ajar). Solusi digitalisasi tata kelola pendidikan.
