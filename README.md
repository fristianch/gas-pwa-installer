# 📑 Sistem Administrasi Biro Jasa (GAS Edition)

[![Google Apps Script](https://shields.io)](https://script.google.com/macros/s/AKfycbyWQGFvvHKVvljYFk6-js7uUbM-H7xRDH37YNT7QLGyTmDC4B8XxPPCPieCmAWXaBQ8/exec)

Aplikasi manajemen operasional Biro Jasa berbasis Google Apps Script. Sistem ini dirancang untuk mempermudah pengelolaan data STNK, SIM, dan BPKB secara online dan terpusat.

## 🔗 Link Aplikasi (Web App)
Aplikasi dapat diakses langsung melalui tautan berikut:
👉 **[Buka Aplikasi Administrasi Biro Jasa](https://script.google.com/macros/s/AKfycbwCybSFu8ZZ5xC9Gd50dz18c6r8drgs6AoD-wasDxrR5DhlvWHuXeA6RyN6HBG5g0rbyA/exec)**

## 🚀 Fitur Utama
*   **Database Terintegrasi:** Penyimpanan data aman menggunakan Google Sheets.
*   **Input Real-time:** Form input cepat untuk pendaftaran berkas pelanggan baru.
*   **Status Tracking:** Monitor progres berkas langsung dari dashboard utama.
*   **Cloud Based:** Dapat diakses dari mana saja tanpa perlu instalasi software tambahan.

## 🛠️ Pengembangan (GitHub Integration)
Repositori ini berisi source code yang disinkronkan dengan Google Apps Script:
*   **Backend:** Logika pemrosesan data di file `.gs`.
*   **Frontend:** Antarmuka pengguna berbasis HTML/CSS di file `.html`.

## 📂 Struktur Project
*   `Code.gs` - Fungsi server-side (CRUD database).
*   `Index.html` - Struktur tampilan Web App.
*   `appsscript.json` - Manifest file untuk konfigurasi scope dan library.

---

**Catatan:** Jika Anda adalah pengembang, pastikan untuk melakukan *Deploy sebagai Web App* dengan akses "Anyone with Google Account" agar aplikasi berjalan sesuai fungsinya.
